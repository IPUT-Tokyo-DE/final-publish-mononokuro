# CAR RACE

## ゲームの内容
車で障害物に当たらないようにして、HPを0にしないようにコースを3週してゴールを目指す

![画面イメージ](docs/images/game_image01.png)

## 操作説明
- SPEACEキーで前に進む
- Aキーで左に回転　Dキーで右に回転
- Qキーで左に45度回転してダメージを受けなくなる　Eキーで右に45度回転してダメージを受けなくなる
